# API 키·활용신청 상태 (2026-07-21 프로브)

기존 공공데이터포털 키(`~/.g2b` 재사용): **유효** (나라장터 200 정상).
아래 API들은 같은 키로 쓰되 **포털에서 API별 활용신청**이 필요 (자동승인, 즉시).

|---|---|---|
| 국토부 아파트 매매 실거래가 (RTMS) | 403 Forbidden | 활용신청 필요 |
| 국토부 아파트 분양권전매 실거래가 | 403 Forbidden | 활용신청 필요 |

## 발급·신청 절차 (사용자 액션)

1. **공공데이터포털 활용신청** — data.go.kr 로그인 → 아래 4개 검색 → 각 페이지에서 [활용신청] (심의 자동승인):
   - "아파트 매매 실거래가 자료" (국토교통부)
   - "아파트 분양권전매 실거래가 자료" (국토교통부)
   - "건축HUB 건축인허가정보 서비스" (국토교통부)
   - "민간아파트 분양가격 동향" (주택도시보증공사)
2. **ECOS**: ecos.bok.or.kr → Open API 서비스 → 회원가입 → 인증키 신청 (즉시 발급)
3. **KOSIS**: kosis.kr/openapi → 회원가입 → 활용신청 → 인증키 발급 (즉시)
4. **R-ONE**(선택): reb.or.kr → R-ONE 부동산통계 → OpenAPI 신청

발급된 키는 `~/개발/config.json`의 `ecos_key` / `kosis_key` / `rone_key`에 기입.
활용신청 완료 후 `python3 src/collect/probe.py` 재실행으로 확인.

## 상가(상권)정보 sdsc2 (2026-07-21 실측, Task 9)

- 베이스 `apis.data.go.kr/B553077/api/open/sdsc2` — 기존 service_key로 **정상 200**.
- `largeUpjongList` 25개 대분류(indsLclsCd A1~U1), `middleUpjongList?indsLclsCd=`로 중분류.
- 시도 필터는 `storeListInDong?divId=ctprvnCd&key=<시도2자리>` (+`indsLclsCd`/`indsMclsCd` 병용 가능,
  `numOfRows=1`로 totalCount만 취득). `storeListInUpjong`은 지역 필터 불가, `storeListInAdmi`는 404.
- 시도코드는 **개편 신코드만 인식**: 강원 42→**51**, 전북 45→**52** (구코드 NODATA_ERROR 03).
- 간헐 502 Bad Gateway 실측(장시간 연속호출 중) → 백오프 재시도로 흡수. 업소 0건은 resultCode 03.
- 수집 완료: 시도17×대분류25 + 서울·경기·부산 중분류(각 91개) → `data/sbiz.json`
  (전국 업소 2,725,313 · 중분류합=대분류합 3개 시도 전부 일치 · 주요상권 CSV 1,227건 통합).

## RTMS 실거래가 수집 결과 (2026-07-21, `src/collect/rtms.py` → `data/rtms.json`)

매매 최근 36개월(202308~202607)·분양권 최근 24개월. 원본 XML 전량 `data/raw/rtms/` 캐시(재호출 스킵).
`dealAmount`(만원)→원 변환 후 ㎡당가 검증범위 [2백만, 5천만] 원/㎡ 밖은 제외.

| 항목 | 결과 |
|---|---|
| 매매 유효 거래 | **201,738건** (제외 16,223 = 강남 >50M/㎡ 519 + 지방 <2M/㎡ 15,704) |
| 분양권 유효 거래 | **15,066건** (제외 28) |
| 커버 시도 | 17개 중 **15개** (광주·전남 제외) |

- **화성시(41590)**: 2025년 구 신설로 통합코드가 빈 응답 → 하위코드 `41591`(만세구)·`41593`(효행구) 등 `41590~41593` 병합 수집(매매 8,809건).
- **광주(29)·전남(46)**: 전 시군구·전 기간(202301~) 매매·분양권 모두 `totalCount=0`(resultCode 000 정상 응답의 빈 items). 강남과 교차 반복호출로 스로틀 아님을 실증 — 서버측 데이터 공백, 대안 시군구 없음(전 구가 0). `sigungu_used`에 note 기록.
- 지방 <2M/㎡ 제외 15,704건은 대부분 실제 저가 노후·소형 아파트(천안 3,686·포항 3,647·전주 3,195 등). 검증범위 하한이 지방장에 공격적 → 해당 시도 median은 ≥2M/㎡ 구간만 반영(상향 편의 주의). 프리미엄 지역(강남·노원·수원영통) 하한 제외 0건으로 만원→원 변환 정합성 실증.
- 간헐 502 Bad Gateway → 5xx/네트워크 백오프 재시도(2·4·6·8s ×5)로 흡수, 캐시-재개.
