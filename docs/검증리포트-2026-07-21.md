## 4. 접근성 상세 (통과 근거)

- **차트 svg**: 모든 주요 차트(line/waterfall/tornado/heatmap/gauge/fan/hbars/phase)에 `role="img"` + 의미있는 `aria-label`.
  라인차트 8종은 호출부(app.js)에서 지역·지표명 aria 전달 확인. 타일 스파크라인은 aria-hidden(장식).
- **입력**: 계산기 range/number 입력에 `<label for>` 연결, number에 `aria-label="… 직접 입력"`. select도 label 연결.
- **키보드**: 테마 토글 `<button>`(2곳), 앱바 탭 `<a>`, 타일 role=button+tabIndex=0+Enter/Space 핸들러 — 모두 tabbable.
- **헤딩 계층**: h1(히어로 1개) → h2(챕터 도곽 제목 8개) → h3(장내 절) → h4/h5. 단일 h1, 논리적 하강.
- **테이블/탭**: ledger·sources·matrix 표에 aria-label, mode-tabs `role="tablist"`.
- **콘트라스트**: `--ink-3`(#99897b) 캡션 vs 지면(#f6f4f1) 스팟 체크 — 캡션 용도로 허용 범위.
